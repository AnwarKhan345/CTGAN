Conditional Table GAN Synthesizer.

This is the core class of the CTGAN project, where the different components are orchestrated together.

For more details about the process, please check the [Modeling Tabular data using Conditional GAN](https://arxiv.org/abs/1907.00503) paper.

  • embedding_dim (int) â Size of the random sample passed to the Generator. Defaults to 128.

  • gen_dim (tuple or list of ints) â Size of the output samples for each one of the Residuals. A Resiudal Layer will be created for each one of the values provided. Defaults to (256, 256).

  • dis_dim (tuple or list of ints) â Size of the output samples for each one of the Discriminator Layers. A Linear Layer will be created for each one of the values provided. Defaults to (256, 256).

  • l2scale (float) â Wheight Decay for the Adam Optimizer. Defaults to 1e-6.

  • batch_size (int) â Number of data samples to process in each step.
